# Кредитный Калькулятор для банковской отрасли

Веб-приложение для автоматизации расчета кредитов с управлением базой клиентов и кредитных продуктов.

## Технологический стек

- **Backend:** Node.js + Express.js
- **Database:** MongoDB + Mongoose
- **Frontend:** EJS Templates + CSS (shadcn/ui style)
- **Runtime:** Node.js v18+

## Функциональность

- Расчет аннуитетных и дифференцированных платежей
- Формирование детального графика платежей
- Управление базой клиентов (CRUD операции)
- Управление кредитами и статусами
- Адаптивный современный интерфейс

## Установка

### 1. Установите MongoDB

Скачайте и установите MongoDB Community Server:
https://www.mongodb.com/try/download/community

### 2. Клонируйте репозиторий

```bash
git clone https://github.com/ahksm/proyektnaya-rabota.git
cd proyektnaya-rabota
```

### 3. Установите зависимости

```bash
npm install
```

### 4. Настройте переменные окружения

Отредактируйте файл `.env`:

```env
ConnectionString=mongodb://localhost:27017/credit_calculator
PORT=3000
```

### 5. Запустите сервер

```bash
npm start
```

Приложение будет доступно по адресу: http://localhost:3000

## Использование

### Калькулятор кредитов

1. Перейдите на страницу "Калькулятор"
2. Введите параметры кредита:
   - Сумма кредита (сум)
   - Годовая процентная ставка (%)
   - Срок (месяцев)
   - Тип платежа (аннуитетный/дифференцированный)
3. Нажмите "Рассчитать"
4. Просмотрите детальный график платежей

### Управление клиентами

- **Добавить клиента:** Заполните форму с данными клиента
- **Просмотр:** Список всех клиентов с возможностью сортировки
- **Редактирование:** Изменение данных существующего клиента
- **Удаление:** Удаление клиента из базы

### Управление кредитами

- **Оформление кредита:** Выберите клиента и параметры кредита
- **Просмотр:** Список всех кредитов с фильтрацией
- **Детали:** Полная информация о кредите с графиком платежей

## Формулы расчета

### Аннуитетный платеж

```
P = S × [i × (1 + i)^n] / [(1 + i)^n - 1]
```

где:
- P – ежемесячный платеж
- S – сумма кредита
- i – месячная процентная ставка
- n – количество месяцев

### Дифференцированный платеж

```
P_n = (S / n) + (Остаток × i)
```

где:
- P_n – платеж в n-й месяц
- S – сумма кредита
- n – количество месяцев
- Остаток – остаток задолженности
